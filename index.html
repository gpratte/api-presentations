API Anatomy

Request / Response

Request

  • http method (verb)
  • protocol (http or https)
  • url
  • port
  • headers
  • body

Example

https://swapi.co/api/people/1/
Bring up api in chrome and show dev tools network 
https://swapi.co/api/people/1/
  • http method: GET
  • protocol: https
  • url: swapi.co/api/people/1/
  • port: 443
  • headers: browser supplied
  • body: none

Response

  • headers
    • status
    • content-type
    • set-cookie
    • date
    • ...
  • content
https://swapi.co/api/people/1/
  • headers
    • status: 200
    • content-type: text/html; charset=utf-8
    • set-cookie: cookie data
    • date: Tue, 13 Feb 2018 13:14:35 GMT
    • ...
  • content: html
